# MPV Binaries (Bundled with Vreamio)

MPV player is bundled with Vreamio for seamless video playback with full audio/subtitle track support.

## Current Version

- Source: shinchiro's mpv-winbuild-cmake
- Build: mpv-x86_64-20260121-git-d6e7fb0

## Files included:

- `mpv.exe` - Main MPV player executable
- `d3dcompiler_43.dll` - Required Direct3D shader compiler

## Updating MPV

To update MPV to a newer version:

1. Download from: https://github.com/shinchiro/mpv-winbuild-cmake/releases
2. Get the latest `mpv-x86_64-*.7z` file (NOT the `dev` version)
3. Extract and replace:
   - `mpv.exe`
   - All `.dll` files
